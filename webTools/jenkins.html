# 前端 Jenkins 自动化部署

# 一.Jenkins

# 1.Jenkins 安装

下载 Jenkins (opens new window). 打开终端进入到下载目录.(先安装好jdk) 运行命令 java -jar jenkins.war --httpPort=8080. 打开浏览器进入链接 http://localhost:8080. 然后就是初始化和使用了 按照说明完成安装.

# 2.Jenkins 配置

进入后会让我们输管理员密码，打开网页上提示路径下的文件，复制密码粘贴输入即可。 然后会让安装需要的插件，此处选默认即可，等待安装完成。 创建一个管理员账户。

注意

由于网络原因(墙)很多插件会下载不下来,非比必要可以暂时不去管,之后在plugin中下载;或者通多更改镜像地址jenkins 下载插件失败 有效的处理办法（亲测） (opens new window)

需要的插件

  • git插件 Git plugin
  • svn插件 Subversion Plug-in 这个jenkins自带已经安装过了，推荐安装时有显示
  • NodeJS
  • Publish Over SSH
